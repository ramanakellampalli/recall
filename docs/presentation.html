Solution

Meet recall

A zero-dependency CLI snippet manager

💾
Save
Store any command with tags & descriptions. Captures git context automatically.
🔍
Find
Fuzzy search with frecency ranking. Filter by tag, repo, or time window.
▶️
Run
Execute directly — no copy, no paste. Prompts for placeholders automatically.
~/.recall/snippets.json  ·  100% local  ·  no cloud

Getting Started

Get started in
10 seconds

# 1. Install globally npm install -g recall-dev # 2. Save your first snippet recall save "docker system prune -af" -t docker -d "Nuclear clean" # 3. Find & run it recall run docker clean
Node.js v18 or later required
recall save
recall find
recall run
recall list

Dynamic Templates

Save templates,
not just commands

Use {placeholders} for dynamic values

# Save a template once recall save "kubectl logs {pod} -n {namespace}" -t k8s # Run it — recall prompts for each variable recall run k8s logs Command: kubectl logs {pod} -n {namespace} {pod}: api-7d9f-xk2p {namespace}: production Running: kubectl logs api-7d9f-xk2p -n production

Smart Search

Find anything
instantly

Fuzzy search + filter DSL + frecency ranking

recall find docker restart # fuzzy search recall find tag:aws deploy # filter by tag recall find repo:payments error # filter by git repo recall find after:1h # recent commands

Results ranked by what you use most, not just keyword match.

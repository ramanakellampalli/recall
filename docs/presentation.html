The Solution
Meet recall
A zero-dependency CLI snippet manager — built entirely for the terminal.
💾
Save
Store any command with tags, a description, and automatic git context — repo, branch, remote all captured.
🔍
Find
Fuzzy search with frecency ranking. Filter by tag, repo, or time window using a simple DSL.
▶️
Run
Execute directly — no copy-paste. Detects {placeholder} variables and prompts for each value before running.
npm install -g recall-dev ~/.recall/snippets.json  ·  100% local  ·  no cloud

Getting Started
Up and running in 10 seconds
01
Install globally
npm install -g recall-dev
Node.js v18 or later required.
02
Save a snippet
recall save "cmd" -t tag -d "desc"
Tags, descriptions and git context all optional but useful.
03
Find & run it
recall run docker clean
Fuzzy-matches your snippet and runs it directly.
# save once recall save "docker system prune -af" -t docker -d "Nuclear clean" # find & run later recall run docker clean

Dynamic Templates
Save templates, not just commands
Wrap variable parts in {curly braces} — recall detects them and prompts for each value at run time.
# save a template once recall save "kubectl logs {pod} -n {namespace}" -t k8s # run — recall prompts for each variable recall run k8s logs Command: kubectl logs {pod} -n {namespace} {pod}: api-7d9f-xk2p {namespace}: production Running: kubectl logs api-7d9f-xk2p -n production

recall-dev 08 / 10
Smart Search
Find anything instantly
Fuzzy search + filter DSL + frecency ranking — results ranked by usage, not just keyword match.
fuzzy

recall find docker restartMatches snippets containing those words in any order, in command or description.

tag:

recall find tag:aws deployFilter to snippets tagged with "aws", then fuzzy-match "deploy" within those.

repo:

recall find repo:payments errorOnly snippets saved while working inside the "payments" git repository.

after:

recall find after:7dSnippets from the last 7 days. Also supports 1h, 30d, and ISO date strings.

💡 Filters stack: recall find tag:k8s repo:payments after:7d deploy
